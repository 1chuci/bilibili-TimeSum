# Bilibili-TimeSum

一个简洁的网页工具，用于快速计算 Bilibili 视频合集或播放列表的总时长。

在您准备“入坑”一个几十集的学习视频或实况全流程视频之前，用它来估算一下总“肝度”吧！

**[➡️ 在线体验]**(https://bilibili-timesum.pages.dev/)

## ✨ 主要特性

*   **一键计算**：快速计算 B 站视频合集的总时长
*   **智能统计**：显示视频数量、平均时长、最长视频等详细信息
*   **便捷操作**：支持一键清空、复制结果、键盘快捷键
*   **实时验证**：输入时自动验证格式，提供友好的错误提示
*   **响应式设计**：完美适配桌面端和移动端，支持深色模式
*   **安全可靠**：内置输入验证和防护机制，确保使用安全
*   **纯净体验**：纯前端实现，界面现代化，无广告
*   **部署轻松**：静态页面，可以轻松部署在任意静态托管平台

## 📦 部署

### Vercel 一键部署

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/your-username/bilibili-timesum)

点击上方按钮即可一键部署到 Vercel。

### Cloudflare Pages 部署

1. 将项目代码推送到 GitHub 仓库
2. 访问 [Cloudflare Pages](https://pages.cloudflare.com) 并登录
3. 点击 "创建项目" → "连接到 Git"
4. 选择您的 GitHub 仓库
5. 配置构建设置：
   - **构建命令**：留空（静态文件无需构建）
   - **构建输出目录**：`/`（根目录）
6. 点击 "保存并部署"

部署完成后会获得一个 `.pages.dev` 域名，也可以绑定自定义域名。

## 🚀 如何使用

### 方法一：使用内置帮助功能（推荐）

1. 打开本工具，点击底部的"**使用帮助**"链接
2. 在弹出的帮助窗口中，点击"**📋 复制代码**"按钮
3. 打开 B 站视频合集页面，按 `F12` 打开开发者工具
4. 点击"控制台"选项卡，粘贴代码并按 `Enter`
5. 复制输出的时长数据，粘贴到本工具中即可

### 方法二：手动复制代码

1. 在浏览器中打开您想要计算的 Bilibili 视频**合集播放页面**
2. 按下 `F12` 键，打开浏览器的"开发者工具"
3. 找到并点击"**控制台**"（Console）选项卡
4. 将下方的代码**完整复制**并粘贴到控制台中，然后按下 `Enter` 键

```javascript
[...document.querySelectorAll('div.stat-item.duration')].map(el => el.textContent.trim()).join('\n')
```

5. 控制台会输出所有视频的时长列表，复制这些数据

### 使用计算器

1. 将复制的时长数据粘贴到本工具的文本框中
2. 点击"计算总时长"按钮或按 `Ctrl+Enter` 快速计算
3. 查看详细的计算结果和统计信息
4. 可以点击"📋"按钮复制计算结果

## 🎯 支持的时间格式

- `MM:SS` - 分:秒（如 `07:33`）
- `HH:MM:SS` - 时:分:秒（如 `01:15:15`）
- 自动忽略无效格式的行
